## 📸 BILDEKATEGORIER OG EKSEMPLER

### **KATEGORI 1: Profesjonelle håndverkere**

**Bruk:** Håndverker-kampanjer (Meta + Google)

**Hva bildet skal vise:**
- Ekte norsk håndverker (ikke stock photo-aktig)
- På jobb, i arbeidsklær
- Godt verktøy synlig
- Smilende, profesjonell
- Norsk setting (gjenkjennelig)

**Tekst-overlay:**
```
🔨 BLI TESTPILOT
Kun 20 plasser

eller

SØKER 20 HÅNDVERKERE
Gratis testing + 40% rabatt
```

**Hvor få bilder:**
- Unsplash.com (søk: "norwegian carpenter", "plumber working")
- Pexels.com
- Ta egne bilder av ekte håndverkere (best!)

**Eksempel Canva-mal:**
```
┌─────────────────────────┐
│                         │
│   [Bilde: Håndverker]  │
│                         │
│   🔨 BLI TESTPILOT     │
│   Kun 20 plasser       │
│                         │
│   ✓ Gratis testing     │
│   ✓ 40% rabatt 6 mnd   │
│                         │
│   [REGISTRER DEG NÅ]   │
│                         │
└─────────────────────────┘
```

---

### **KATEGORI 2: Fornøyde kunder i renovert hjem**

**Bruk:** Kunde-kampanjer (Meta + Google)

**Hva bildet skal vise:**
- Norsk familie/person
- Nyoppusset baderom, kjøkken, stue
- Moderne, fint hjem
- Smil, tilfredshet
- Før/etter-kontrast (valgfritt)

**Tekst-overlay:**
```
TRENGER HÅNDVERKER?
Få 3-5 tilbud på 24 timer

eller

VI FANT PERFEKT ELEKTRIKER
På 1 dag med Håndtverkeren
```

**Eksempel Canva-mal:**

### **KATEGORI 3: Mobilskjerm (app mockup)**

**Bruk:** Begge kampanjer (Meta + Google)

**Hva bildet skal vise:**
- Mobiltelefon med Håndtverkeren-app
- Skjerm viser:
  - Jobbtilbud (for håndverkere)
  - Eller tilbud fra håndverkere (for kunder)
- Moderne UI
- Håndtverkeren-logo

**Tekst-overlay:**
```
MOTTA JOBBER DIREKTE
På mobilen - enkelt og trygt

eller

SAMMENLIGN TILBUD
Velg beste håndverker
```

**Verktøy for app mockups:**
- Canva (har ferdige iPhone/Android-templates)
- Figma (mer avansert)
- MockuPhone.com (gratis)

---

### **KATEGORI 4: Badges/sertifikater**

**Bruk:** Eksklusivitets-kampanjer

**Hva bildet skal vise:**
- Premium badge-design
- "TESTPILOT #1-20"
- "Eksklusivt medlemskap"
- Gull/blå farger
- Offisiell følelse

**Tekst-overlay:**
```
EKSKLUSIVT TESTPILOT-MERKE
Bli en av de 20 første
```

**Eksempel design:**

### **KATEGORI 5: Før/etter transformasjon**

**Bruk:** Kunde-kampanjer (viser verdi)

**Hva bildet skal vise:**
- Split-screen: før og etter
- Dramatisk forskjell

## 🎬 VIDEO-FORMATER

### **Meta Ads Video:**

**Lengde:** 15-30 sekunder (maks 60 sek)

**Størrelser:**
- Kvadrat: 1080x1080 px (1:1) ⭐
- Stående: 1080x1920 px (9:16) - Stories
- Liggende: 1920x1080 px (16:9)

**Format:** MP4, MOV

**Filstørrelse:** Maks 4 GB

**Viktig:**
- ✅ Inkluder undertekster (folk ser uten lyd!)
- ✅ Fang oppmerksomhet i første 3 sekunder
- ✅ Logo synlig hele tiden
- ✅ Tydelig CTA i slutten

---

### **YouTube Ads (valgfritt):**

**Lengde:** 
- Skippable: 15-60 sekunder
- Non-skippable: 15-20 sekunder
- Bumper: 6 sekunder

**Størrelse:** 1920x1080 px (16:9)

**Format:** MP4

## 🖼️ CANVA-TEMPLATES (Bruk disse!)

### **Template 1: Håndverker-rekruttering (1080x1080)**

**Lag i Canva:**

1. **Åpne Canva** → Velg "Instagram Post" (1080x1080)
2. **Bakgrunn:** 
   - Gradient: #17384E → litt lysere blå

3. **Tekst:**
   - Topp: "🔨 HÅNDTVERKEREN" (hvit, Inter Bold, 48px)

     ```
     ✓ Gratis testing
     ✓ 40% rabatt i 6 mnd
     ✓ Eksklusivt merke
     ```
     (hvit, Inter Regular, 24px)

   - Form: Rundet rektangel
   - Farge: #E07B3E
   - Tekst: "REGISTRER DEG NÅ" (hvit, Inter Bold, 28px)
   - Størrelse: 400x80 px

   - "fagfolk6.vercel.app/pilot" (hvit, opacity 80%, 18px)

## 🎯 BEST PRACTICES

### **1. Følg 20%-regelen (Facebook)**

❌ FEIL: Mer enn 20% av bildet er tekst
- Facebook reduserer reach
- Høyere CPC

✅ RIKTIG: Maks 20% tekst
- Test med Facebook Text Overlay Tool
- Mindre tekst = bedre reach

---

### **2. Mobiloptimalisering**

✅ SJEKK:
- Tekst er lesbar på liten skjerm (min. 24px)
- CTA-knapp er stor nok (min. 80x80 px)
- Viktigste info i midten (ikke kanters)

---

### **3. Konsistens**

✅ ALLTID:
- Samme farger (#17384E + #E07B3E)
- Samme fonts (Inter)
- Samme logo-plassering
- Samme stil

→ Bygger merkevare-gjenkjennelse

---

### **4. A/B-testing av design**

Test:
- Ulike farger på CTA-knapp
- Med/uten emoji
- Ekte bilder vs. illustrasjoner
- Lyse vs. mørke bakgrunner

La data bestemme!

## ✅ DESIGN-SJEKKLISTE

**Før du publiserer:**

- [ ] Riktig størrelse (1080x1080 for Meta, 300x250 for Google, etc.)
- [ ] Farger er korrekte (#17384E, #E07B3E)
- [ ] Font er Inter (eller Arial backup)
- [ ] Tekst er lesbar på mobil
- [ ] CTA-knapp er tydelig
- [ ] Logo er synlig
- [ ] Maks 20% tekst i bilde (Facebook)
- [ ] Eksportert i høy kvalitet (PNG, 300 DPI)
- [ ] Bildet er relevant for målgruppen
- [ ] Ingen copyright-problemer (bruk gratis stock eller egne bilder)
