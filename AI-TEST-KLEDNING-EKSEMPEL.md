## 📊 **BEREGNET MATERIALE:**

### **For 12m x 8m vegg med STÅENDE kledning:**

**Input:**
- Lengde: 12 meter
- Høyde: 8 meter
- Areal: 96 kvm

**Beregning:**
```
Boardbredde: 10 cm (0.10m) - standard for stående kledning
Gap mellom bord: 1 cm (0.01m)
Effektiv bredde per bord: 0.11m

Antall bord: 12m / 0.11m = 109.09 ≈ 110 stk
Lengde per bord: 8m
Lineære meter: 110 × 8m = 880m
Med 15% svinn: 880m × 1.15 = 1012m

✅ RESULTAT: Ca. 110 bord à 8m = 1012 lineære meter
```

---

### **Hvis det var LIGGENDE kledning:**

**Beregning:**
```
Boardbredde: 12 cm (0.12m) - standard for liggende kledning
Overlap: 2 cm (0.02m)
Effektiv høyde per bord: 0.10m

Antall bord: 8m / 0.10m = 80 stk
Lengde per bord: 12m
Lineære meter: 80 × 12m = 960m
Med 15% svinn: 960m × 1.15 = 1104m

✅ RESULTAT: Ca. 80 bord à 12m = 1104 lineære meter
```

## 💡 **FREMTIDIGE FORBEDRINGER:**

### **1. Budsjett-estimat for kledning**

Legg til i `estimateBudget()` funksjon:

```typescript
if (category === 'trevare' && answers.workType === 'Kledning/panel (vegg/fasade)') {
  const sizeMatch = answers.wallSize.match(/(\d+)\s*m?\s*x\s*(\d+)\s*m?/i);
  if (sizeMatch) {
    const area = parseFloat(sizeMatch[1]) * parseFloat(sizeMatch[2]);
    const materialCost = area * 200; // 200 kr/kvm for materiale
    const laborCost = area * 400; // 400 kr/kvm for arbeid
    const min = materialCost + laborCost;
    const max = min * 1.3; // +30% for usikkerhet
    
    return {
      min: Math.round(min),
      max: Math.round(max),
      message: `Basert på ${area} kvm kledning`
    };
  }
}
```
