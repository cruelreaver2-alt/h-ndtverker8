## 🧊 **2. ISOLASJON - HVORFOR VIKTIG?**

### **Bestemmer direkte hvilken effekt som trengs!**

**Formel:**
```
Effektbehov (kW) = Areal (kvm) × Effektfaktor (W/kvm)

Effektfaktor varierer med isolasjon:
• Svært godt isolert: 30-40 W/kvm
• Godt isolert: 40-50 W/kvm
• Moderat isolert: 50-60 W/kvm
• Dårlig isolert: 60-80 W/kvm
```

---

### **EKSEMPEL: 150 kvm hus**

#### **Svært godt isolert (nybygg-standard)** ✅

## 📏 **3. TAKHØYDE - HVORFOR VIKTIG?**

### **Påvirker volum som skal varmes!**

**Formel:**
```
Volum (m³) = Areal (kvm) × Takhøyde (m)

Effektbehov baseres på VOLUM, ikke bare areal!
```

---

### **EKSEMPEL: 35 kvm stue**

#### **Standard takhøyde (2,4 - 2,7 meter)** ✅
```
Volum: 35 kvm × 2,5 m = 87,5 m³

Anbefalt pumpe: 2,5 kW

KOSTNAD: 25 000 - 30 000 kr (normalt)

MONTERING:
→ Standard høyde
→ Enkel montering
→ Ingen ekstrakostnad
```

---

#### **Høyt tak (over 2,7 meter)** ⚠️
```
Volum: 35 kvm × 3,2 m = 112 m³ (+28% volum!)

Anbefalt pumpe: 3,0-3,5 kW (+15-20%)

KOSTNAD: 28 000 - 35 000 kr (+15%)

MONTERING:
→ Må monteres høyere
→ Kan kreve lift/stillas
→ Ekstrakostnad montering: +2 000 - 5 000 kr

TOTALT: +3 000 - 8 000 kr ekstrakostnad
```

---

#### **Lavt tak (under 2,4 meter)** ✅
```
Volum: 35 kvm × 2,2 m = 77 m³ (-12% volum)

Anbefalt pumpe: 2,0-2,5 kW

KOSTNAD: 22 000 - 28 000 kr

MONTERING:
→ Enklere montering
→ Mindre effekt nødvendig
→ SPART: 3 000 - 5 000 kr 🎉
```

## 🧮 **AI SMART BUDSJETT-BEREGNING:**

### **Multiplier-system:**

```typescript
function calculateHeatPumpBudget(
  sqm: number,
  insulation: string,
  propertyAge: string,
  ceilingHeight: string
): { min: number; max: number } {
  
  // Base price for air-to-air heat pump
  let baseMin = 30000;
  let baseMax = 80000;
  
  // 1. INSULATION MULTIPLIER
  let insulationMultiplier = 1.0;
  if (insulation === 'Svært godt isolert') {
    insulationMultiplier = 0.8; // -20% (smaller pump needed)
  } else if (insulation === 'Godt isolert') {
    insulationMultiplier = 0.9; // -10%
  } else if (insulation === 'Dårlig isolert') {
    insulationMultiplier = 1.3; // +30% (larger pump needed)
  }
  
  // 2. PROPERTY AGE MULTIPLIER
  let ageMultiplier = 1.0;
  if (propertyAge === 'Gammel (over 50 år)' || 
      propertyAge === 'Eldre (30-50 år)') {
    ageMultiplier = 1.2; // +20% (electrical upgrades)
  }
  
  // 3. CEILING HEIGHT MULTIPLIER
  let ceilingMultiplier = 1.0;
  if (ceilingHeight === 'Høyt (over 2,7 meter)') {
    ceilingMultiplier = 1.15; // +15% (more volume)
  }
  
  // TOTAL MULTIPLIER
  const totalMultiplier = 
    insulationMultiplier * ageMultiplier * ceilingMultiplier;
  
  // CALCULATE FINAL BUDGET
  const min = Math.round(baseMin * totalMultiplier);
  const max = Math.round(baseMax * totalMultiplier);
  
  return { min, max };
}
```
